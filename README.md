## 🧠 设计原则（第一性原理）

### 1. 系统与策略严格解耦
- **系统** = 市场环境模拟器（只提供数据、规则、执行）
- **策略** = 交易决策主体（完全独立，自主决策）
- 系统层绝不干预：交易决策、止损止盈、仓位管理

### 2. 时间同步
- ❌ 不使用本地系统时间判断交易
- ✅ 以数据流的时间戳为准

### 3. T+1 规则
- 今日买入的股票 → 明天才能卖
- 区分：总持仓 vs 可用持仓

### 4. 真实滑点
```
滑点 = 市场冲击 + 延迟损耗 + 买卖价差
```
- 小盘股流动性差时滑点可达 0.5%-1%

### 5. 交易成本
- 佣金：万 2.5（最低 5 元）
- 印花税：卖出千 1
- 过户费：万分之 0.1（最低 1 元，双向）

## 🛡️ 风险提示

⚠️ **重要认知**:
- 模拟盘 ≠ 实盘
- 模拟盘可以验证策略逻辑
- 模拟盘不能模拟真实的心理承受力
- "模拟盘赚 100 万，实盘可能亏 10 万"

建议：先用小仓位（≤10% 资金）实盘验证！
